  • X-Ray Scans for Pneumothorax Detection ( Feb-2018 )
    A ChestXRay dataset, X-ray labeled with corresponding diagnoses to build a model that classifies patients with a Pneumothorax i.e collapsed lung. To visualize why the network made that decision, Class Activation Mapping (CAM) technique is used which generates "heatmaps" indicating the regions of the image to which model "attends" in the last layer.
